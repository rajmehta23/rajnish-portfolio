## ✨ What is this?
This is my interactive resume and portfolio website. I built this to showcase my journey as a BCA Merit Holder and my skills in web development. Instead of just sending a boring PDF resume, I wanted recruiters and clients to *experience* my work!

### Highlights:
* **Always Updated:** I can add new projects using a hidden admin dashboard.
* **Interactive Design:** Smooth animations and a "glass" aesthetic that looks great on laptops and phones.
* **Virtual Assistant:** A built-in chat window to quickly help visitors learn about my skills.

---

## 🚀 How to Publish This to GitHub Pages (1-Click Deploy)

I have configured this project to be incredibly easy to host for free on GitHub Pages!

1. **Upload your code** to a GitHub repository named `rajnish-portfolio`.
2. **Open your computer's terminal**, go to this folder, and type:
   ```bash
   npm install
   npm run deploy
   ```
3. **That's it!** In a few minutes, your website will be live at: `https://[your-username].github.io/rajnish-portfolio/`

*(Note: In your GitHub Repo Settings, go to "Pages" and ensure the source is set to the `gh-pages` branch).*

## 💻 For Developers (Under the Hood)
If you're a developer checking out my code, here's what I used to build this:
* **Frontend:** React 19, Tailwind CSS
* **Animations:** Framer Motion & GSAP
* **Backend:** Firebase (for secure login and database storage)

To run it locally on your computer:
```bash
npm install
npm run dev
```
