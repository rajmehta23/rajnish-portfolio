# Rajnish Kumar - Portfolio

A high-performance, responsive personal portfolio demonstrating modern web development capabilities, interactive UI design, and full-stack integration.

![Portfolio Preview](https://images.unsplash.com/photo-1555066931-4365d14bab8c?auto=format&fit=crop&q=80&w=1200&h=400)

## 🚀 Features

*   **Modern Interactive UI:** Built with React 19, Framer Motion, and GSAP for fluid, scroll-driven animations and a glassmorphism aesthetic.
*   **Virtual Assistant Interface:** Includes a custom-built interactive chat widget that guides users through the portfolio and answers questions dynamically.
*   **Real-time Admin Dashboard:** Secured via Firebase Authentication, allowing real-time database management of portfolio projects and incoming messages.
*   **Fully Responsive:** Designed mobile-first using Tailwind CSS to look pixel-perfect on any device.

## 🛠️ Tech Stack

*   **Frontend:** React (v19), Vite, Tailwind CSS, shadcn/ui
*   **Animations:** Framer Motion, GSAP, Three.js
*   **Backend & DB:** Firebase (Auth, Firestore)
*   **Routing:** React Router DOM

## ⚙️ How to Run Locally

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/rajmehta23/rajnish-portfolio.git
    cd rajnish-portfolio
    ```

2.  **Install NPM packages**
    ```bash
    npm install
    ```

3.  **Setup Environment Variables**
    Rename `.env.example` to `.env` and enter your API keys.

4.  **Run the development server**
    ```bash
    npm run dev
    ```
