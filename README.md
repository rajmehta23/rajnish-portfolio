## ✨ What is this?
This is my interactive resume and portfolio website. I built this to showcase my journey as a BCA Merit Holder and my skills in web development. Instead of just sending a standard PDF resume, I wanted recruiters and clients to *experience* my work!

### 🚀 Highlights:
* **Always Updated:** I add and manage new projects dynamically via a hidden admin dashboard hooked up to a real-time database.
* **Interactive Design:** Smooth animations and a sleek aesthetic that looks pixel-perfect on both desktop and mobile screens.
* **Virtual Assistant:** A custom-built, interactive chat interface to quickly help visitors learn about my technical skills and background.

## 🛠️ Tech Stack Built With:
* **Frontend:** React 19, Vite, Tailwind CSS, shadcn/ui
* **Animations:** Framer Motion & GSAP
* **Backend:** Firebase (Authentication & Firestore Database)
* **Routing:** HashRouter (Optimized specifically for GitHub Pages CI/CD)
